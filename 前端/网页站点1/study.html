个人网站 - 学习内容

  • 主界面
  1. 1

    学习 HTML

    掌握 HTML 基础标签，如段落标签、标题标签、列表标签等。学习 HTML5 新增特性，包括语义化标签,通过实际项目练习，熟练运用 HTML 构建网页结构。

  2. 2

    学习 CSS

    深入学习 CSS 选择器、盒模型、浮动、定位等基础概念。掌握 CSS3 新增特性，如过渡、动画、弹性布局和网格布局。能够使用 CSS 对网页进行美化和布局设计，实现响应式设计，使网页在不同设备上都能有良好的显示效果。

  3. 3

    学习 JavaScript

    学习 JavaScript 基础语法，包括变量、数据类型、函数、控制语句等。了解 DOM 操作，能够通过 JavaScript 动态修改网页内容和样式。掌握事件处理机制，实现网页的交互效果。学习异步编程，如回调函数、Promise、async/await 等。

  4. 4

    学习 Vue.js

    了解 Vue.js 的基本概念，如组件化开发、响应式原理等。掌握 Vue.js 的核心语法，包括模板语法、计算属性、监听器等。学习 Vue Router 实现单页面应用的路由功能，使用 Vuex 进行状态管理。通过实际项目实践，熟练运用 Vue.js 构建前端应用。

  5. 5

    学习 Spring Boot

    了解 Spring Boot 的自动配置原理，快速搭建 Spring Boot 项目。掌握 Spring Boot 与数据库的集成，如使用 Spring Data JPA 进行数据库操作。学习 Spring Boot 的 RESTful 接口开发，实现前后端的数据交互。掌握 Spring Boot 的日志管理、配置管理等功能。

  6. 6

    前后端结合开发

    学习如何将 Vue.js 前端项目与 Spring Boot 后端项目进行整合，实现前后端分离开发。掌握跨域问题的解决方案，确保前后端数据交互正常。学习使用 Axios 等工具进行前后端的数据请求和响应处理。通过实际项目实践，掌握前后端结合开发的流程和技巧。

  7. 7

    坚持在 CSDN 发布博客

    定期总结学习过程中的知识点和经验，撰写技术博客发布到 CSDN 平台。通过博客分享，加深对知识的理解和掌握，同时提高自己的技术表达能力和影响力。积极参与 CSDN 社区的交流和互动，与其他开发者共同进步。

  8. 8

    学习新项目和新技术

    关注行业动态，学习新的前端和后端技术，如 React、Angular、微服务架构等。参与开源项目，学习优秀的代码结构和设计模式。通过学习新项目和新技术，不断拓宽自己的技术视野，提升自己的技术水平。
